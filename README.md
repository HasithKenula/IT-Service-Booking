# TechShield LK - Professional Laptop Repair & Tech Support Website

This is a complete, multi-page website for a laptop repair and tech support business in Sri Lanka.

## Features
- **Brand Name**: TechShield LK
- **Modern Design**: Built with Tailwind CSS, Glassmorphism, and smooth animations.
- **Booking System**: Users can book repairs (Laptop, Phone, etc.). Data is saved to your browser's Local Storage for demo purposes.
- **Technician Dashboard**: specific dashboard for technicians to view and manage bookings. It includes charts and KPI cards.
- **Multiple Pages**: Home, Services, Booking, Contact, Dashboard.

## How to Run
1. Simply double-click on `index.html` to open the website in your browser.
2. Navigate through the menu links.
3. Try booking a repair in `booking.html`.
4. Go to `dashboard.html` to see your booking appear in the system!

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- FontAwesome (Icons)
- Google Fonts (Outfit)
- Chart.js (Dashboard Analytics)

## Note
This project does NOT require Node.js. It runs directly in the browser.
